## Tool Proficiencies

If your character does not have any tool proficiencies, feel free to skip this part. Weapon and armour proficiencies are taken care of automatically, this is just for things like musical instruments, thieves' tools, vehicles and so on.

Our Downtime system replicates different professions, and does give bonuses for proficiency in relevant tools, but you will need to set up those tool proficiencies for them to be counted. You may also find they come in handy during games!

Head to [ğŸ¤–bot-spam](https://discord.com/channels/866376531995918346/866397299437600778), and for each tool you are proficient in, run `!tool pro <toolname>`.

For example:
- `!tool pro Lute`
- `!tool pro Thieves Tools`

If you have tool expertise, instead run `!tool exp <toolname>`

You can see a full list of available tools with `!tool all`, if in doubt, and can check your current proficiencies with `!tool list`.

### If you have any doubts or questions:

- For syntax, check `!tool help`
- For what counts as a proficiency, and whether this is something you need to do, tag a Helper
- If you think the system is not working as it should, tag a Dragonspeaker
